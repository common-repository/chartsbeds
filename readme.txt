== Description ==
Chartsbeds Review is a powerful tool from ChartsPMS. You can use it, to show statistic of your hotel/hostel on a website. Reviews of your guests and main statistic of your hotel can be shown as a widgets or on a page using shortcode.

== Installation ==
* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Plugin Settings and fill settings.

== Frequently Asked Questions ==
How to use plugin?

Shortcodes
[chartsbeds-review-circle] - to activate circle review statistics

[chartsbeds-review-bar] - to activate bar review statistics

[chartsbeds-review-page] - to activate reviews on page. Has limit settings.

Limits

With [chartsbeds-review-recent] and [chartsbeds-review-page] you can use limits. Example [chartsbeds-review-page limit="8"]


Widgetsgit 

Chartsbeds recent reviews widget
Chartsbeds bar statistics widget


Adminpanel

Key field - unique field for property that recieves data from chartsPMS. 

Insert Reviews page url - url of the Review page on your site. When you put some data, in Recent widget appears button with link. If you dont have, leave empty. (optional)

Recent reviews widget, choose amount to show : - Choose how many comments to show in Recent reviews widget.( from 2 to 8 and 4 by default).

Check to disable gravatars for reviews widget - if checked, gravatars from Recent reviews widget will not appear.

Check to disable hotel's answer for reviews - if checked, hotel's answers will be hidden.
